# House Prices Prediction (Kaggle Competition)

Predicting house prices using XGBoost on the Ames Housing dataset.

## Overview
- Exploratory Data Analysis with visualizations
- Feature engineering and preprocessing
- XGBoost model training
- Kaggle submission generated

## Results
- Distribution of prices and top correlated features visualized
- Submission file included (upload to Kaggle for score)

## Files
- house_prices.ipynb - Full code notebook
- submission.csv - Predictions
- price_distribution.png - Price histogram
- top_correlations.png - Top features

This project demonstrates an end-to-end data science workflow.
